Embeddable Widget Demo

This page simulates how the standalone widget behaves on a plain HTML site. Update the workspace id and API base URL to match your local environment.

Embed snippet <script src="http://localhost:3000/widget.js" data-workspace-id="REPLACE_WITH_WORKSPACE_ID" data-api-base-url="http://localhost:8000" data-position="right" data-theme="light" data-color="#0ea5e9" data-welcome-message="Hi there. Ask about pricing, onboarding, or policies." ></script>

The widget uses Shadow DOM isolation, session persistence, streaming chat, inline lead capture, and public workspace settings sync.
